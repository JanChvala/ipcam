Android IP camera

About - O projektu

English

This page represents sample project which is used for demonstration the principles described in my master thesis. The goal was to create Android IP camera application with the ability to show the camera content from the web page. The application has to work in even if the Android device is behind Network Address Translator (NAT).

Web Real Time Communication (WebRTC), Google Cloud Messaging and AngularJS are the main implementation pillars. You can read the whole master thesis to find out more.

To start first download the APK, install it on Android device running Android 5.0+ and call the Android IP camera using the Floating Action Button at the righ bottom corner of this screen.

Česky

Tato stránka představuje jednoduchý projek, který slouží pro demonstraci principů popsaných v mojí diplomové práci. Cílem práce bylo vytvořit Android IP kamera, aplikaci která dokáže přenést multimediální data z videokamery zařízení a přehrát je pomocí webové stránky. Aplikace musí být schopná pracovat v reálných podmínkách a to i v případě, že je zařízení schované za překladačem síťových adres (NAT).

Web Real Time Communication (WebRTC), Google Cloud Messaging a AngularJS jsou hlavními implementačními nástroji. Pro více informací můžete nahlédnout do celého znění diplomové práce.

Abyste mohli zobrazit videokameru z telefonu, nejprve stáhněte a nainstalujte APK balíček do Android zařízení běžící na Androidu ve verzi 5.0+. Potom můžete zahájit hovor pomocí FAB (Floating Action Button( tlačítka nacházejícího se v pravo dola na této stránce
